# MTK-SusSPACE
Universal AnyKernel with deep SuSFS integration for Kernel 5.10+ devices. Designed for MediaTek, UNISOC, Exynos &amp; Tensor chips where custom kernels are restricted. Bypasses strict app root-detection completely while keeping 100% original stock kernel stability, performance, and zero bugs. Works seamlessly with ReSukiSU root manager.
